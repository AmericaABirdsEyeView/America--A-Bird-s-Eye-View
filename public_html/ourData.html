Our Data

Unfortunately, the web server we are using only allows us to host 1GB of data, so we aren't able to put all of our pre-processed data online. However, we are more than happy to share with you our processed data.


Approval Ratings

Our approval ratings data is located here. The file names are in the following format mm_dd_yyyy.txt. Inside the text file, the file formatting is pretty straight forward: state-name|number-of-positive-tweets|number-of-negative-tweets with the data for one state per line.


Top Tweets

Our top tweets data is located here. The file names are in the following format mm_dd_yyyy.txt. Inside the JSON file, is the JSON format of a tweet, as returned by the Twitter API. The tweets are sorted in ascending order by popularity.


Top URLs

Our top URLs data is located here. The file names are in the following format mm_dd_yyyy.txt. Inside the text file, there are two pieces of information for each of our top URLs. First, the URL itself, then a "||", followed by the title of the page that the URL points too, as provided by the Decent URL Title API , then a new line. The URLs are in ascending order, sorted by popularity.


Top Hashtags

Our top hashtags data is located here. The file names are in the following format mm_dd_yyyy.txt. Inside the text file, the hashtags are sorted in ascending order by popularity, and there is one hashtag per line.
